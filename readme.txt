== Description ==

If caused the error in the theme editing, management screen allows viewing.

If you set in the User Settings screen, it can starting when only the user has logged in.
It will not work theme on the management screen.
Error messages are not displayed.
Confirmation of the error message I done in the public side.

== Installation ==

Installation using "Add New Plugin"
1. From your Admin UI (Dashboard), go to Plugins. Click the "Add New" button.
2. Search for plugin, or click the "Upload" button, and then select the plugin you want to install.
3. Click the "Install Now" button.

Manual installation
1. Upload the "theme-update-mode" folder to the "/wp-content/plugins/" directory.

Settings
1. visit "Edit My Profile" or "User Edit"
2. check the "theme update mode"
